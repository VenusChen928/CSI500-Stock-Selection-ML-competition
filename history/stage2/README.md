# Stage2 History

Archived Stage2 experiments and records.  The active Stage2 code and final
portfolio are kept at the repository root and under `submissions/stage2/`.

## Contents

- `scripts/`: inactive but still informative model experiments.
- `rejected_scripts/`: clearly rejected directions.
- `submissions/`: old generated portfolios.
- `backtests/`: old per-window generated portfolios.
- `reports_archive/`: old score tables, sweeps, leakage scans, and worklogs.
- `EXPERIMENT_LOG.md`: chronological decision notes.

Use these files for report evidence, not for the final upload.
