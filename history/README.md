# History Layout

This folder archives files that are useful for report writing or reproducibility
but are not needed in the active Stage2 submission root.

## Folders

- `common/`: generic old tools, backups, unused data, IDE config, and legacy
  reports.
- `stage1/`: Stage1-specific scripts, submissions, and reports.
- `stage2/`: Stage2 experiments, old generated portfolios, rejected scripts,
  and archived score reports.

Nothing in this folder is required for uploading `portfolio.csv`.
