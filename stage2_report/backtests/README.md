# Stage2 Backtest Portfolios

This folder contains the per-window portfolio CSVs generated by the final
12-window Stage2 self-test.

The main regenerated run is:

```text
stage2_report/backtests/guard_route_v2_nocache_fullweek12_20260510/
```

Portfolio CSVs are actual submitted-shape portfolios for one historical as-of
date and one model route, such as `baseline_xgb_20260116.csv` or
`baseline_guard_adaptive_20260116.csv`.  Files named
`baseline_guard_adaptive_meta_*.csv` are companion route diagnostics from the
adaptive model, not upload candidates.  The aggregate evidence lives in:

```text
stage2_report/final_report_materials/
```

To regenerate these portfolio records and refresh the summary/detail files,
run the full-week self-test command from the root `README.md` or
`final_report.md`.
