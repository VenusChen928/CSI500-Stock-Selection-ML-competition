# Submissions Layout

This is the submission-facing folder.

Upload exactly:

```text
submissions/portfolio.csv
```

## Active

- `portfolio.csv`: final Stage2 portfolio CSV.
- `stage2/final_report_materials/`: organized evidence for the Stage2 written
  report.

## Archived Elsewhere

- Stage1 submissions: `history/stage1/submissions/`.
- Old Stage2 generated portfolios: `history/stage2/submissions/`.
- Final duplicate portfolio copies:
  `history/stage2/submissions/final_duplicate_portfolios_20260510/`.
- Old Stage2 backtest portfolios: `history/stage2/backtests/`.
- Generic legacy reports: `history/common/submissions_legacy/`.
